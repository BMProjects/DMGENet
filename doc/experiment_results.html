★ 洞察：动态权重策略

RLMC 学习到的权重具有强烈的步长依赖性：
• 1h: N (37.5%) + S (30.5%) + POI (32.0%) 三者均衡，D 图几乎不贡献 (0.08%)
• 6h: N 图主导 (59.5%)，显著领先
• 12h: N (43.4%) + POI (56.3%) 形成互补，S 和 D 几乎忽略
• 24h: 四图更均衡，N 仍最强 (39.7%)，D 贡献上升至 9.0%
这种自适应权重正是 RLMC 相较于静态平均集成的核心优势。

★ 关键发现：Delhi 在长程预测上反而更优

在 h=12 和 h=24 时，Delhi NCT 的 RMSE 均低于 Beijing（46.38 vs 48.11；52.08 vs 61.42）。这一反直觉结果说明：Delhi 的污染源更集中（工业+地形），月/季节周期性更强，GNN 空间图结构能更好地捕捉这种规律性。这正面回应了 R1 的泛化质疑。
